# AI Agent 学习仓库

> 一份系统化的 AI Agent 领域学习指南，从入门到精通的 30 天学习计划

## 📚 仓库概览

本仓库是一份完整的 **AI Agent（人工智能代理）** 学习资源库，旨在帮助学习者系统地理解和掌握 AI Agent 的核心概念、关键技术、主流框架和实践应用。

### 目录结构

```
learn-ai-agent/
├── README.md                          # 仓库说明（您在这里）
├── knowledge-map.md                   # AI Agent 领域知识地图
├── concepts.md                        # 核心概念与技术拆解
├── AGENT.md                          # AI Agent 核心原理与架构
├── learning-plan-30-days.md          # 30 天学习路线图
├── progress.md                        # 学习进度记录
└── daily/                            # 每日学习任务
    ├── 01.md ~ 30.md                # 30 天具体学习计划
    └── ...
```

## 🎯 学习目标

完成本学习计划后，你将能够：

1. **理论基础**
   - 理解 AI Agent 的定义、分类与发展阶段
   - 掌握 Agent 的核心架构与工作原理
   - 了解感知、决策、执行的完整流程

2. **技术栈**
   - 学习主流 Agent 框架（LangChain、AutoGPT、CrewAI 等）
   - 理解 LLM、检索增强（RAG）、工具调用等关键技术
   - 掌握 Agent 与环境交互的方式

3. **实践应用**
   - 能够使用框架开发简单 Agent
   - 理解多 Agent 协作的设计模式
   - 学会 Agent 的测试、调试与优化

4. **前沿发展**
   - 了解 Agent 在不同领域的应用案例
   - 掌握当前研究热点与趋势
   - 建立个人学习与实践框架

## 📖 如何使用本仓库

### 快速开始

1. **了解全貌** → 阅读 [knowledge-map.md](./knowledge-map.md)
2. **掌握概念** → 学习 [concepts.md](./concepts.md)
3. **制定计划** → 查看 [learning-plan-30-days.md](./learning-plan-30-days.md)
4. **按天学习** → 依次学习 `daily/01.md` ~ `daily/30.md`
5. **查阅参考** → 遇到问题时参考 [AGENT.md](./AGENT.md)
6. **记录进度** → 在 [progress.md](./progress.md) 记录学习成果

### 推荐学习方式

- **自学模式**：按照每日任务逐步学习，每天花费 1-2 小时
- **项目驱动**：每周完成一个小项目，巩固本周学习内容
- **讨论学习**：与他人讨论遇到的问题，加深理解
- **笔记整理**：主动记录要点，建立个人知识体系

## 💡 学习建议

### 时间投入
- **基础自学**：每天 1-2 小时，坚持 30 天
- **加深理解**：每周安排一个完整项目或案例实战
- **持续学习**：完成后保持对新框架、新技术的关注

### 学习方法
1. **理论 + 实践相结合**：先理解概念，再动手实现
2. **由浅入深**：从简单 Agent 开始，逐步构建复杂场景
3. **主动反思**：记笔记、写总结、进行知识重构
4. **社区交流**：参加讨论、阅读他人作品、互相反馈

### 资源获取
- 优先学习官方文档和主流框架文档
- 查阅最新的学术论文和行业分析报告
- 关注 GitHub 上的优秀开源项目
- 订阅相关技术公众号和 RSS 源
