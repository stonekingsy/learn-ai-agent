## 🎓 核心概念

### 什么是 Agent？

**简单定义**：Agent 是一个能够感知环境、制定计划、自主执行并从结果中学习的自主程序。

**关键特征**
1. **感知（Perception）** - 获取环境信息
2. **决策（Decision）** - 基于目标和信息进行思考
3. **执行（Action）** - 与环境交互
4. **学习（Learning）** - 从反馈改进

### Agent vs 传统程序

| 维度 | 传统程序 | AI Agent |
|------|--------|---------|
| 控制流 | 线性、按顺序执行 | 目标驱动、循环执行 |
| 决策 | 预定义规则 | 动态推理、LLM 决策 |
| 环境交互 | 被动响应请求 | 主动感知和探索 |
| 适应性 | 固定逻辑 | 可学习和优化 |
| 知识来源 | 硬编码的规则 | LLM + 知识库 + 工具 |

### 当前 Agent 浪潮的驱动力

```
LLM 的出现（GPT-3, GPT-4, Claude）
        ↓
具有强大的理解和推理能力
        ↓
催生了新一代 Agent（LLM 驱动）
        ↓
无需写大量规则，靠提示词指导
        ↓
Agent 成为通用的智能系统
```

## ❓ 常见问题

**Q: Agent 和 Chatbot 有什么区别？**
A: Chatbot 是被动回复，Agent 是主动做事。Agent 有目标、计划和工具调用能力。

**Q: 学习 Agent 需要什么基础？**
A: 需要了解 Python 编程，对 LLM 有基本认知最好。

**Q: Agent 现在能做什么实际的事情？**
A: 可以做任务自动化（爬虫、数据处理）、内容生成、决策辅助等。
