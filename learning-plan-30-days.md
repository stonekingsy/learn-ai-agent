# 30 天 AI Agent 学习路线

> 系统化、阶段性的学习计划，从入门到掌握 AI Agent

## 📋 学习计划概览

### 学习阶段划分

```
阶段一（第 1-5 天）：基础认知
├─ 目标：理解 Agent 的定义、历史、核心概念
├─ 重点：建立正确的认知框架
└─ 成果：能解释"什么是 AI Agent"

阶段二（第 6-10 天）：核心技术基础
├─ 目标：深入理解 LLM、Prompt、工具调用
├─ 重点：掌握 Agent 的技术基础
└─ 成果：理解工具调用、Prompt 工程基础

阶段三（第 11-15 天）：关键技术深化
├─ 目标：学习 RAG、记忆管理、链式推理
├─ 重点：掌握让 Agent 更智能的关键技术
└─ 成果：能设计简单的记忆系统

阶段四（第 16-20 天）：框架实战入门
├─ 目标：学习 LangChain、构建第一个 Agent
├─ 重点：从理论到实践的转变
└─ 成果：能构建一个简单的工具使用 Agent

阶段五（第 21-25 天）：框架深化与多 Agent
├─ 目标：掌握框架高级用法，理解多 Agent 系统
├─ 重点：提升工程能力，理解协作模式
└─ 成果：能构建多 Agent 系统

阶段六（第 26-30 天）：综合应用与优化
├─ 目标：应用所学知识做项目，了解前沿发展
├─ 重点：巩固学习，展望未来
└─ 成果：完成一个完整的 Agent 应用项目
```
